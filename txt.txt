🎯 Общая информация
Название: Подарок в большом городе
Тип проекта: Интернет-магазин эксклюзивных изделий ручной работы
Технологический стек: Python 3.11, Django 4.2, Bootstrap 5, SQLite, Gunicorn

✨ Функциональные возможности
1. Главная страница (/)
    Приветственный баннер (Jumbotron)
    Кнопки быстрого перехода в каталог и на форму связи
    Карточки преимуществ магазина
    Анимированные элементы
2. Каталог товаров (/goods/)
    Отображение всех товаров с изображениями
    Поиск по названию и описанию товаров
    Фильтрация по категориям
    Пагинация (6 товаров на странице)
    Карточки товаров с кнопками "Подробнее" и "Заказать"
3. Карточка товара (/goods/product/<id>/)
    Детальное описание товара
    Изображение товара в увеличенном формате
    Хлебные крошки для навигации
    Секция "Похожие товары" (из той же категории)
    Кнопка заказа товара
4. Страница "О нас" (/aboutas/)
    История магазина
    Преимущества работы с нами
    Адаптивная двухколоночная верстка
5. Страница "Подробнее" (`/readmore/)
    Описание категорий изделий:
    Декорирование предметов
    Съедобные букеты
    Миниатюрные домики
6. Форма заявки (/forms/)
    Поля: Имя, Email, Телефон, Сообщение
    AJAX-отправка без перезагрузки страницы
    Валидация формы на стороне клиента и сервера
    Модальное окно успешной отправки
    Автоматический редирект на главную через 3 секунды
    Telegram-уведомления о новых заявках

🎨 Особенности дизайна
Стилизация
Цветовая схема: Фиолетовый градиент (#667eea → #764ba2)
Шрифт: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
Анимации: FadeIn, SlideInUp при загрузке карточек
Sticky Footer: Фиксированная навигация внизу экрана
Bootstrap Icons: Иконки в интерфейсе

Локальный запуск (Development)
# 1. Клонирование репозитория
git clone <репозиторий>
cd pbc-shop

# 2. Создание виртуального окружения
python -m venv env
source env/bin/activate  # Linux/Mac
# или
env\Scripts\activate     # Windows

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Применение миграций
python manage.py migrate

# 5. Сбор статики
python manage.py collectstatic --noinput

# 6. Запуск сервера
python manage.py runserver

Деплой на Amvera
# 1. Создание приложения в Amvera
amvera create

# 2. Настройка переменных окружения в amvera.yml
# Добавить:
# TELEGRAM_BOT_TOKEN=your-bot-token
# TELEGRAM_CHAT_ID=your-chat-id

# 3. Деплой
amvera push

🔐 Безопасность
CSRF Protection — защита от межсайтовой подделки запросов
XSS Protection — фильтрация вредоносного кода
SQL Injection Protection — ORM Django защищает от инъекций
Password Validation — валидация паролей пользователей
Secure Headers — настройки безопасных заголовков
